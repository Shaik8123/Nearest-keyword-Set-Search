# Nearest Keyword Set Search

A web-based search system designed to retrieve the nearest matching keyword sets from multidimensional datasets.

## Technologies Used

* JSP
* HTML
* CSS
* JavaScript
* SQL
* Apache Tomcat

## Features

* Keyword-based search
* Dataset retrieval
* User authentication
* Search result analysis
* Database integration

## Project Structure

* Source Code
* Database Scripts
* UI Components

## How to Run

1. Import project into Eclipse.
2. Configure Apache Tomcat.
3. Import database.sql into MySQL.
4. Run the application.
